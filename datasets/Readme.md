## 1. mat file

The .mat files are the opinion scores of different datasets.

ODI_enhance.mat : IQA-ODI

ODV_dmos_sorted.mat : ODV-VQA

JVQD2.mat : JVQD

360SVQD.mat : 360SVQD

The opinion scores are sorted in ascending order.
